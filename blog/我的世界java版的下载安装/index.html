我的世界java版的下载安装

发表于2022-08-06|更新于2022-08-06

这次讲一下mc的java版下载与安装啊

hmcl下载地址 提取码:e536

下载完成之后去java官网下载java
java官网

安装完java之后打开hmcl就可以下载版本了
没有正版账号的就选离线模式(下图)

下载版本点击安装源下载自己喜欢的版本
在设置里设置java路径然后打开刚安装的java文件夹打开bin再点击java.exe
注意1.7.2及以下版本需要java7.0不能太高的版本，java7.0可以自行去网上下载
如果要加mod需要安装froge，如果mod有要求froge版本就选择那个版本下

如果想要添加人物皮肤可以去Little skin下载
Little skin

点开那个衣架的图标然后选择本地皮肤文件，最后把你下载的皮肤打开就好了

如果想要联机首先确保双方游戏版本相同模组相同
房主先打开存档然后打开对局域网开放创建一个世界

完成后回到hmcl启动界面打开多人联机点击创建房间

最后把邀请码发给朋友等待他进入房间即可
